# Student Activity Log Analyzer

## Overview
This Python program analyzes student activity logs and generates a report with:
- Student-wise activity summary
- Daily activity statistics
- Detection of abnormal behavior (multiple logins without logout)

---

## Concepts Used
- Python OOP
- Regular Expressions
- Generators
- File Handling
- Dictionaries (`defaultdict`)
- Exception Handling

---

## Input Format (`student_log.txt`)
Each log entry contains student ID, name, activity, date, and time.


---

## Output
- `activity_report.txt` containing summaries and statistics
- `output_report.txt` containing the total number of logins and submissions

---

## How to Run
```bash
   python student_activity.py
```
